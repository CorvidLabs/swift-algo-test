# Changelog

## [0.1.0] - 2025-12-02

### Added
- Sandbox management with `LocalSandbox` actor and `Sandbox` protocol
- Account management with `AccountFactory`, `AccountPool`, and `FundedAccount`
- Transaction building with `TestTransactionBuilder` fluent API
- Pre-built transaction scenarios via `TransactionScenarios`
- Mock clients: `MockAlgodClient` and `MockIndexerClient`
- State snapshots with `SnapshotCapture` and `SnapshotStore`
- Comprehensive XCTest assertions for Algorand testing
- Full Swift 6 and strict concurrency support
- Sendable conformance on all public types
